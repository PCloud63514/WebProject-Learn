# Expo 해상도 UI 대응



### Dimensions

> Device의 너비와 높이를 확인할 수 있는 모듈.

 

예제

```js
import { Dimensions, View, Text, StyleSheet } from 'react-native'

```

### Platform

> 플랫폼(Android or IOS)을 감지하는 모듈.

### PixelRatio

> Device의 픽셀 밀도(Pixel Density)와 폰트 스케일(font scale)을 감지하는 모듈.
>
> - 픽셀 밀도: 1인치당 픽셀의 수
