# AOP(Aspect Oriented Programming)?

AOP는 **관점 지향 프로그래밍**을 의미합니다. 

관점 지향은 어떤 로직을 기준으로 핵심적인 관점, 부가적인 관점으로 나누어 보고 각 관점을 기준으로 모듈화(공통된 로직 및 기능을 하나의 단위로 묶는 것)를 

하는 방법입니다.
